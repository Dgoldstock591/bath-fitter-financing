# Bath Fitter Synchrony Financing Planner

Single-file web app for quick, in-home financing quotes (960, 925, 965, 924, 922).

- True amortization math
- Rounded-up monthly payments
- Mobile-friendly design
- Integrated “Sales Tips”
- Branded for Bath Fitter
- Created by **David Goldstock**

## How to use
1. Open the site (via GitHub Pages).
2. Enter a job total.
3. Press **Calculate** to see instant plan comparisons.
